# Loopwish · Windows

![Loopwish banner](https://raw.githubusercontent.com/loopwish/shared/main/assets/logos/loopwish-banner.svg)

Loopwish er en ønskeliste-app som lar deg dele ønsker med familie og venner. Ønsk. Del. Få. Sammen.

Dette repoet inneholder Windows-klienten for Loopwish.

## Plattformstøtte

- Windows 10+

## Teknologi-stack

- .NET 8
- WPF
- xUnit

## Kom i gang

### Forutsetninger

- .NET SDK 8.x
- Windows (for å kjøre WPF-appen)

### Bygg og test

```bash
dotnet build -c Release
dotnet test -c Release
```
